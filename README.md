# Mon Projet JavaFX

## Installation
1. Installer JDK 17+
2. Cloner le projet : `git clone https://github.com/DylanCAD/env-java.git`
3. Importer dans IntelliJ IDEA

## Commandes utiles
- `mvn javafx:run` → Lancer l'application
- `mvn test` → Exécuter les tests
- `mvn checkstyle:check` → Vérifier la syntaxe
